Keys stay in your browser only. They go straight to OpenAI or Anthropic when you run a tool — not through Socrates servers.
New to API keys? What they are & how to get one

An API key is not your Socrates password. It’s a secret code you create at OpenAI or Anthropic so your computer can use their AI. There is usually a small pay‑as‑you‑go cost (often pennies per run) paid to them, not to Socrates.

OpenAI (keys start with sk-, not sk-ant): create an account → Sign up → API keys → “Create new secret key” → paste here.

Claude (Anthropic) (keys start with sk-ant-): console.anthropic.com → API keys → create → paste here.
